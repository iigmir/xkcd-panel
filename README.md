# xkcd-panel

A pure API application that can receive JSON data from [https://xkcd.com](https://xkcd.com), with a simple SPA(Single Page Application).

## API list

* `/api/v1/xkcd`
    Method: GET
    Get latest xkcd comic.
* `/api/v1/xkcd/:number`
    Method: GET
    Get specific number of xkcd comic.
